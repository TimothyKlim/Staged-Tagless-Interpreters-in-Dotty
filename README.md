# Staged-Tagless-Interpreters-in-Dotty

This project is following two papers : 
 - "A Gentle Introduction to Multi-Stage programming" by Walid Taha
 - "Finally Tagless, Partially Evaluated" by Jacques Carette, Oleg Kiselyov and Chung-chieh Shan
 
You can find the paper about tagless interpreters at http://okmij.org/ftp/tagless-final/JFP.pdf and the code associated to it at http://okmij.org/ftp/tagless-final/incope.ml.
